# 🤝 Как помочь CAOS

Спасибо, что хотите помочь платформе коллективного целеполагания!

## 🐛 Нашли баг?

1. Проверьте, нет ли такого [Issue](https://github.com/thethinkred-ai/caos-platform/issues)
2. Если нет — создайте новый с описанием:
   - Что делали
   - Что ожидали
   - Что произошло
   - Скриншот/логи (если есть)

## 💡 Есть идея?

Создайте Issue с пометкой `enhancement`. CAOS — молодой проект, любая идея ценна.

Особенно интересны:
- Улучшение онтологии целей
- Интерфейсы для новых типов участников
- Интеграции с другими платформами
- Предложения по процедурам коллективного целеполагания

## 🛠 Хотите написать код?

### Первый раз на GitHub?

Пошаговая инструкция в README — раздел **«Как помочь проекту»**.

### Локальная разработка

```bash
docker compose up --build
```

Без Docker — можно запустить компоненты по отдельности.

### Быстрый чеклист перед Pull Request

- [ ] Докер собирается без ошибок (`docker compose up --build`)
- [ ] API-тесты проходят (`pytest`)
- [ ] Фронтенд компилируется (`cd frontend && npm run build`)
- [ ] Название коммита понятное
- [ ] Описали в PR, что сделали и зачем

### Стиль кода

- **Python:** FastAPI-стиль, type hints, докстринги
- **TypeScript:** строгая типизация, функциональные компоненты
- Комментарии на русском или английском

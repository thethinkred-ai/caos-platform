# CAOS Roadmap

## Foundation: weeks 1–2

Vision, PRD, ontology, architecture, ADR, coding standards, module passports and acceptance criteria.

## Kernel: weeks 3–4

Authentication, users, problems, goals, projects, tasks and the 30-day vertical slice.

## Alignment: weeks 5–6

Decision history, goal alignment, search, integrity rules and optional graph read-model spike.

## Activity memory: weeks 7–8

Teams, actions, resources, audit, notifications, project results and knowledge.

## Practice and AI: weeks 9–10

Versioned practices and four bounded AI operations with provider fallback.

## Pilot: weeks 11–12

Production deployment, security/accessibility/E2E checks, restore drill and 20–50 user pilot.

## Post-MVP

Evaluate graph scaling, capability/opportunity graph, mobile, federation and advanced AI only after pilot evidence.
